10. Diferencie:
- nó raiz
- nó folha
- nó interno

    O nó raiz é onde se inicia a árvore e ele não tem pai, o nó folha já é o fim da árvore
e ele não possui filhos, já o nó interno é o meio termos dos outros dois pois ele possui
pelo menos um filho e tem um pai.

11. Explique o conceito de ancestral e descendente de árvores.

    O ancestral é qualquer nó que aparece antes de outro nó no caminho até a raiz. 
Já descendente é qualquer nó que está abaixo de um determinado nó na árvore.

12. O que é profundidade de um nó?

    É a quantidade de níveis entre o nó e a raiz da árvore.

13. Explique a diferença entre grau de um nó e grau de uma árvore.

    O grau de um nó é a quantidade de filhos que um nó possui, já o de uma árvore é 
a maior quantidade de filhos que um único nó possui na estrutura.

14. Qual a importância das árvores para sistemas computacionais modernos?

    É importante porque permitem organizar, armazenar e buscar informações de forma eficiente.

    int dado;

    struct No *proximoIrmao;

No* criarNo(int valor) {
    No *novo = (No*) malloc(sizeof(No));

    if (novo != NULL) {
        novo->dado = valor;
        novo->primeiroFilho = NULL;
        novo->proximoIrmao = NULL;
    }

    return novo;
}

int main() {

    printf("Raiz: %d\n", raiz->dado);

    free(raiz);
    return 0;
}

18. Implemente uma função para inserir filhos em um nó.

No* criarNo(int valor) {
    No *novo = (No*) malloc(sizeof(No));

    if (novo != NULL) {
        novo->dado = valor;
        novo->primeiroFilho = NULL;
        novo->proximoIrmao = NULL;
    }

    return novo;
}

void inserirFilho(No *pai, int valor) {
    if (pai == NULL)
        return;

    No *novoFilho = criarNo(valor);

    if (pai->primeiroFilho == NULL) {
        pai->primeiroFilho = novoFilho;
    }
    else {
        No *aux = pai->primeiroFilho;

        while (aux->proximoIrmao != NULL) {
            aux = aux->proximoIrmao;
        }

        aux->proximoIrmao = novoFilho;
    }
}

19. Implemente uma função que conte quantos nós existem em uma árvore.

int contarNos(No *raiz) {
    if (raiz == NULL)
        return 0;

    int total = 1; 

    No *filho = raiz->primeiroFilho;

    while (filho != NULL) {
        total += contarNos(filho);
        filho = filho->proximoIrmao;
    }

    return total;
}

20. Implemente uma função recursiva para calcular a altura de uma árvore.

int altura(No *raiz) {
    if (raiz == NULL)
        return 0;

    int maiorAltura = 0;
    No *filho = raiz->primeiroFilho;

    while (filho != NULL) {
        int h = altura(filho);

        if (h > maiorAltura)
            maiorAltura = h;

        filho = filho->proximoIrmao;
    }

    return maiorAltura + 1;
}

21. Faça uma função que conte quantas folhas existem em uma árvore.

int contarFolhas(No *raiz) {
    if (raiz == NULL)
        return 0;

    if (raiz->primeiroFilho == NULL)
        return 1;

    int total = 0;
    No *filho = raiz->primeiroFilho;

    while (filho != NULL) {
        total += contarFolhas(filho);
        filho = filho->proximoIrmao;
    }

    return total;
}

22. Implemente uma função que verifique se um valor existe na árvore.

int buscarValor(No *raiz, int valor) {
    if (raiz == NULL)
        return 0;

    if (raiz->dado == valor)
        return 1;

    No *filho = raiz->primeiroFilho;

    while (filho != NULL) {
        if (buscarValor(filho, valor))
            return 1;

        filho = filho->proximoIrmao;
    }

    return 0;
}

23. Crie uma função que imprima os nós por nível.

void imprimirNivel(No *raiz, int nivel) {
    if (raiz == NULL)
        return;

    if (nivel == 1) {
        printf("%d ", raiz->dado);
    } else {
        No *filho = raiz->primeiroFilho;

        while (filho != NULL) {
            imprimirNivel(filho, nivel - 1);
            filho = filho->proximoIrmao;
        }
    }
}

void imprimirPorNivel(No *raiz) {
    int h = altura(raiz);

    for (int i = 1; i <= h; i++) {
        imprimirNivel(raiz, i);
        printf("\n");
    }
}

24. Explique por que a recursão é muito utilizada em árvores.

    Porque cada nó pode ser considerado a raiz de uma subárvore formada por seus filhos, então 
um mesmo procedimento pode ser aplicado ao nó atual e, em seguida, repetido para cada uma de suas subárvores.

25. Implemente uma função para destruir/liberar toda a árvore da memória.

void destruirArvore(No *raiz) {
    if (raiz == NULL)
        return;

    No *filho = raiz->primeiroFilho;

    while (filho != NULL) {
        No *proximo = filho->proximoIrmao;

        destruirArvore(filho);

        filho = proximo;
    }

    free(raiz);
}
